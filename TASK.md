We must use the **real Allure HTML report** generated by `(reporter/generate-html-report! results-dir report-dir)` — the same function used in CI.

```clojure
  ;; Scroll down in the overview to capture test details below the fold

```

The 3 screenshots are:
1. `s1` — overview at top (summary stats, report title, pass rate)
2. `s2` — overview scrolled down 600px (test details, metadata)
3. `s3` — overview scrolled down 1200px (more tests if visible)

Use `(core/navigate page url)` + `(page/scroll page ...)` + `(page/screenshot page path)` pattern.
