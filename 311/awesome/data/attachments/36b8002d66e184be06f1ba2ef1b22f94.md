## PUT /echo → 200 OK

### Request Headers
```
PUT /echo
```

### Request Body
```json
{
  "name": "Alice Updated",
  "email": "alice2@example.com"
}
```

### Response Headers
```
content-length: 92
content-type: application/json
date: Sun, 01 Mar 2026 10:11:54 GMT
```

### Response Body
```json
{
  "method": "PUT",
  "path": "/echo",
  "body": {
    "name": "Alice Updated",
    "email": "alice2@example.com"
  }
}
```

### cURL
```bash
curl '/echo' \
  -X PUT \
```
